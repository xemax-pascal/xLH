# SPS Einführung nach IEC-61131-3

Die nachfolgenden Videosequenzen ermöglichen einen schnellen Einstieg in die SPS 
<a href="https://plcopen.org/iec-61131-3" target="_blank">
   IEC61131-3
</a>
Entwicklungsumgebung
<a href="https://de.codesys.com/das-system/warum-codesys.html" target="_blank">
   CODESYS
</a>. Die CODESYS Plattform unterstützt aktuell ca. 1'000 unterschiedliche Industriesteuerungen und wird 
in über 1'000'000 Geräten jährlich eingesetzt   
(<a href="https://de.codesys.com/das-system/codesys-inside.html" target="_blank">
   Geräte- und Herstellerverzeichnis
</a>).

Die direkte Integration eines SPS-Simulators in die Entwicklungsumgebung ermöglicht einen Einstieg ohne konkrete
Hardware. Aus didaktischer Sicht ermöglicht dies einen direkten und schnellen Einstieg in die Algorithmik.  
In der Videoserie können die ersten drei Videos im Windows-internen SPS-Simulator bearbeitet werden 
(Bedienoberfläche, FUP KOP ST CFC und Visualisierung von Prozessvariablen).   
Die weiteren Lernsequenzen erfordern den xLH_base Controller.
