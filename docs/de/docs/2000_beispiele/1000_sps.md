# SPS Einführung nach IEC-61131-3

Die folgenden Videosequenzen bieten einen schnellen Einstieg in die SPS-Entwicklungsumgebung 
<a href="https://plcopen.org/iec-61131-3" target="_blank">
   IEC61131-3
</a>. 
Die CODESYS-Plattform wird von über 500 Herstellern implementiert, in etwa 1.000 verschiedenen Industriesteuerungen
konkretisiert und jährlich in über 1.000.000 Geräten eingesetzt  
(<a href="https://de.codesys.com/das-system/codesys-inside.html" target="_blank">
   Hersteller- und Geräteverzeichnis
</a>).

Die direkte Integration des SPS-Simulators in die Entwicklungsumgebung ermöglicht einen Einstieg ohne konkrete
Hardware. Aus didaktischer Sicht kann damit ein direkter und schneller Einstieg in die Algorithmik umgesetzt werden.  
In der Videoserie sind die ersten drei Videos im Windows-internen SPS-Simulator umgesezt  
(Bedienoberfläche, FUP KOP ST CFC und Visualisierung von Prozessvariablen).   
Die weiteren Lernsequenzen erfordern den xLH_base Controller.
